# Example App

This example demonstrates how to use the `save_points_snackbar_dialog_bottomsheet` package.

## Running the Example

1. Install dependencies:
   ```bash
   flutter pub get
   ```

2. Run the app:
   ```bash
   flutter run
   ```

## Features Demonstrated

- **Dialogs**: Info, Success, Confirmation, Error, Async Loading, Animated, and Circular Reveal
- **Snackbars**: Basic, Success, Error, Warning, Top Position, Gradient, Bounce, Slide Rotate, Tap to Dismiss, Bordered, Elastic, Custom Direction, and Circular Reveal
- **Bottom Sheets**: Basic, Options Menu, Animated, Loading, Scrollable, Settings, and Circular Reveal
